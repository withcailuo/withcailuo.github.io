Linux文件分布

发表于 2020-04-02

User Programs(binaries)

/bin - Essential user commands

contains essential commands that every user will need.

includes your login shell and basic utilities like ls.

usually fixed at the time you install Linux.

/usr/bin - Most user commands

contains the program binaries.

if you just installed a software package and don;t know there the binary went, this is the first place to look.

/usr/local/bin - “local” commands

when you compile software from source code, those install files are usually kept separate from those provided as part of your Linux distribution.

System Administration Programs (binaries)

/sbin - Essential System Admin Commands

contains programs needed by the system administrator, like fsck, which is used to check file systems for errors.

is populated when you install your Linux system, and rarely changes.

/usr/bin - Non-essential System Administration Programs (binaries)

commands for optional system services and network servers.

/usr/local/sbin - “Local” System Administration Commands

when you compile servers or administration utilities from source code, this is where the binaries normally will go.

Libraries

Libraries are shared bits of code.

three directories where library files are placed: /lib, /usr/lib, and /usr/local/lib.
